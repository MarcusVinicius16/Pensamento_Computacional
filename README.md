## 📌 Sobre o Repositório

Este repositório consolida os artefatos desenvolvidos na disciplina de **Pensamento Computacional**, ministrada pela **Profa. Kadidja Valéria**, no curso de Análise e Desenvolvimento de Sistemas (UDF).

O objetivo é demonstrar, de forma prática, a aplicação de fundamentos essenciais da computação na resolução de problemas e na construção de sistemas com foco em escalabilidade, organização e clareza estrutural.

## 🧩 Abordagem Técnica

O desenvolvimento deste repositório está fundamentado nos pilares do Pensamento Computacional:

- **Decomposição:** segmentação de problemas complexos em componentes menores  
- **Reconhecimento de Padrões:** reutilização de soluções consolidadas  
- **Abstração:** simplificação de cenários mantendo o essencial  
- **Algoritmos:** construção de fluxos lógicos estruturados  

---

## 🎯 Objetivo

Este portfólio tem como finalidade:

- Evidenciar a evolução no raciocínio computacional  
- Demonstrar capacidade de modelagem de sistemas  
- Aplicar boas práticas de organização e documentação de código  
- Servir como base para projetos futuros mais complexos
